About Me

I am a 22 year old graduate from Nottingham Trent University with interests in machine learning, data analysis and web development. I have experience in various high level languages such as Python, C++, SQL and Java for applications like creating store systems and processing user data.

I first became interested in computing and technology from the age of 7 after having problems with a Windows 95 computer and watching an IT technician install Ubuntu and then use various terminal commands to navigate and create folders as well as fix the computer. This interest then escalated as I gained knowledge about hardware as well as software through an optional computer science course which allowed me to complete an AS level in it at GCSE level.

Past Experiences

As part of one of my modules at Nottingham Trent University, I was part of a team to create a PoS system for the university staff resources building, this included creating a mobile phone application to scan QR codes so that items could be easily added to a user's basket as well as easy admin visualisations with stock and profit graphs.

Also, due to my interest in poker, I created a joinable game of poker using Java so that my friends could play along with me without having to pay any rake, or commission, to any third party meaning we could play simply with additional features like running the rest of the cards even though everyone has folded.

Hobbies

In addition to my interest in poker, I also play Guitar Hero at a semi professional level using a custom version of it that includes all the games and can be done on computer instead of consoles as the originals were. It also means that custom songs can be loaded as well as the ability for it to be streamed which I do via Discord to my friends.

From my interest in Guitar Hero, I discovered that I also wanted to play guitar and so found my dads old electric guitar in the attic and have been practicing and learning how to play for the past two years.
